# 🧮 Calculadora Pensional Simplificada 🇨🇴

Este proyecto es una herramienta educativa desarrollada como parte de la Actividad 1 (Aprendizaje Basado en Proyectos - ABP) para la asignatura de **Matemática Fundamental** en la **UCompensar (2026)**.

La aplicación permite calcular y proyectar escenarios pensionales básicos en Colombia (Ley 100 de 1993 y Ley 2381 de 2024), aplicando conceptos matemáticos de primer semestre como:
- Aritmética básica
- Proporcionalidad y porcentajes
- Expresiones algebraicas (Función suelo)
- Radicación (Raíz cuadrada para cálculo de riesgo)
- Logaritmos (Base 10 para esfuerzo acumulado)
- Ecuaciones Lineales (Bonus: Rentabilidad de CDT)

### 🚀 Cómo ejecutar el proyecto localmente
1. Clona este repositorio.
2. Instala las dependencias: `pip install -r requirements.txt`
3. Ejecuta la aplicación: `streamlit run calculadora_pension_streamlit.py`
